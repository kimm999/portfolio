# Portfolio - React + Vite

React와 Vite를 사용한 포트폴리오 웹사이트입니다.

## 설치 및 실행

```bash
npm install
npm run dev
```

## LiquidEther 컴포넌트

`src/components/LiquidEther.jsx` 파일에 사용자가 제공한 전체 LiquidEther 구현 코드를 포함해야 합니다.

현재는 기본 구조만 포함되어 있으므로, 사용자가 제공한 전체 React 코드를 `src/components/LiquidEther.jsx`에 복사해주세요.
